# 🔄 Data Transformer - SQL Project

## 📌 Description
Data Transformer is a SQL-based project that demonstrates advanced data manipulation and transformation techniques using relational databases.

It covers joins, subqueries, window functions, string operations, date functions, and conditional logic using real-world datasets.

## 🎯 Objective
The objective of this project is to perform data transformation and analysis using SQL queries commonly used in real-world applications and interviews.

## 🧱 Database Structure
The project includes three main tables:
- Customers (customer details)
- Orders (transaction records)
- Employees (company staff details)

## 🚀 Features
- Create and manage relational database
- Insert and manipulate data
- Perform multiple types of joins:
  - INNER JOIN
  - LEFT JOIN
  - RIGHT JOIN
  - FULL JOIN (using UNION)
- Use subqueries for filtering
- Apply date functions (YEAR, MONTH, DATEDIFF, FORMAT)
- Perform string operations (CONCAT, UPPER, LOWER, REPLACE, TRIM)
- Use window functions (RANK, RUNNING TOTAL)
- Apply CASE statements for categorization and discount calculation

## 🛠️ Technologies Used
- SQL (MySQL)
- Relational Database Design
